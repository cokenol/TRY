  1. Excel Settings
    1. Configuring the settings to use “EChart 21013.xlsm” echart so it doesn`t auto calculate the whole echart. VERY IMPORTANT IF YOU DO NOT WANT TO EXCEL TO LAG.
  2. Import E-Chart from Senai
    1. Open up both “EChart 210113.xlsm” and echart received from Senai, for this example, senai echart is “EChart 210113 210825.xlsm"
    2. When you opened up both echart make sure to press “Don’t update” button. Updating will cause all formulas to recalculate and eventually cause the excel to freeze.
    3. Press Alt + F11 to open up the Microsft Visual Basic for Applications window.
    4. Click the 1. Import Senai Button and the immediate window in the right window should start to show its actions of importing.
    5. Next, we need to update the CIS status of each panels in the database table from the echart. To do this press the “2. Update CIS status” button. This process may take about 20minutes.
    6. Then, we need to check date errors in the echart so when we import there is no errors halting the import process. To do this, press “3. Check Date Error” button.

      7. If the immediate window doesn’t show any “ERROR AT:” messages then it should be fine to go to next step which is importing all the cast and delivery dates.

    7. Finally, now we will be doing the final step to import all the cast, DO number and delivery dates. After completing steps 2.4 till 2.6, now you can press the “4. Import all new dates” button. This process also takes about 20 minutes.
  3. Update on Daily Report
    1. First we will go to this worksheet and to get update on Daily Cast Panels.
    2. Key in the cast date i.e yesterday to set the date to find what has been casted and press the “Get Casted” button to obtain daily cast panels.
    3. Next, go to “CESSmry (3)” worksheet and press Shift + F9 to calculate only that worksheet to update the summary numbers.
    4. Next, go back to workshee “DailyCasted3A4C” and press “Lau Format ALL BLK” to update the casted and delivered panels for each block. This process will take about 2 minutes.
    5. Next, update slab cast date colours in “CESSmry (3)”, if casted it will update as yellow as per worksheet “MCProgram”.
    6. And finally, press the button “Print Report FOR SEAN” to print out the daily cast report.
  4. To print out “pending” to monitor urgent and upcoming deliveries.
    1. Simply press the button in “Check cast& delivery 3”, and it will generate out the urgent & upcoming deliveries in worksheet “Pend”.
    2. Next, copy the previous word file into a new date, and open that file.
    3. Delete the existing table and its content in the word file.
    4. Copy the excel table in “pend” worksheet and paste in the word file.
    5. Adjust the table so it fits to the whole margin of the paper size.
    6. You may go ahead and delete these parts after you checked out the amount of pending to cast. I delete these cuz the pending to cast column will be too cluttered.
    7. You can use ctrl+q to bold the whole line.
    8. You can use ctrl + w to highlight to whole line in yellow if you wish to do so.
    9. You can also use ctrl + e to hightlight in red if you wish to highlight anything in red.
    10. After you are done with editing this pending report, you may then print this in PDF format and share in the whatsapp group.
