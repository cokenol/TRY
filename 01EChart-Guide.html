PEC48 E-Chart guide to use R1

  1. Senai Echart
    1. Configuring the settings to use “EChart 21013.xlsm” echart so it doesn’t auto calculate the whole echart. VERY IMPORTANT IF YOU DO NOT WANT TO EXCEL TO LAG.
  2. Import E-Chart from Senai
    1. Open up both “EChart 210113.xlsm” and echart received from Senai, for this example, senai echart is “EChart 210113 210825.xlsm"
    2. When you opened up both echart make sure to press “Don’t update” button. Updating will cause all formulas to recalculate and eventually cause the excel to freeze.
    3. Press Alt + F11 to open up the Microsft Visual Basic for Applications window.
    4. Click the 1. Import Senai Button and the immediate window in the right window should start to show its actions of importing.
    5. Next, we need to update the CIS status of each panels in the database table from the echart. To do this press the “2. Update CIS status” button. This process may take about 20minutes.
    6. Then, we need to check date errors in the echart so when we import there is no errors halting the import process. To do this, press “3. Check Date Error” button.

      7. If the immediate window doesn’t show any “ERROR AT:” messages then it should be fine to go to next step which is importing all the cast and delivery dates.

    7. Finally, now we will be doing the final step to import all the cast, DO number and delivery dates. After completing steps 2.4 till 2.6, now you can press the “4. Import all new dates” button. This process also takes about 20 minutes.
